# 订单模块

####  描述
*该模块和订单相关，包括订单相关表的创建，订单操作流程(创建订单、查询订单、取消订单、支付订单、删除订单...)* 

-----
#### 订单相关表
-  __order_master__  **订单原始表**
-  __orders__  **订单表** (主表) 
-  __order_appraise__   
-  __order_details__   
-  __order_logistics__   
-  __order_logs__   
-  __order_payment_logs__   
-  __order_products__   
-  __order_returns__   
-  __order_returns_apply__   
-  __order_trades__    
    
-----
#### 功能:
    > 订单创建
    > 订单支付    
    > 订单查询    
    > 订单取消
    > 订单删除
